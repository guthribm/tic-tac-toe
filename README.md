# tic.tac.toe
a react demo

## Overview

This is a game of tic tac toe created with ReactJS and is based off of the <a href="https://reactjs.org/tutorial/tutorial.html" rel="noopener noreferrer" target="_blank">tutorial provided by the reactjs.org</a> documentation. It has been slightly customized and features new UI and animations. It is built with: - HTML - CSS - JavaScript ES6 - create-react-app
